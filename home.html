About


The (Core) Model Viewer is a resource for the exploration, comparison, and analysis of central carbon core metabolic models.

The Model Viewer also serves to demonstrates tools and visualizations to be released in the ModelSEED 2.0

Utilized Data Sources

Much of the data included in the Model Viewer is dervied from numerous published manuscripts and databases. We've done our best to cite contributing data sources here. If you beleive your data has been utilized and not properly cited, please notify us at chenry@mcs.anl.gov. We sincerely thank all of our data sources, without whom this resource would not exist.
